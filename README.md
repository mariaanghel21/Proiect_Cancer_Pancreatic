# Pancreatic Cancer Detection CAD 

A lightweight Computer-Aided Diagnosis (CAD) system for detecting pancreatic cancer in abdominal CT scans. This project uses a custom-trained **YOLOv8 Nano** model integrated into a fast, interactive web interface built with **Streamlit**.

##  Key Features
* **High Accuracy:** Achieved a 92.6% mAP50 on independent validation data.
* **Fast Inference:** Runs in under 100ms per CT slice on a standard CPU.
* **User-Friendly UI:** Easy to upload scans, adjust confidence thresholds, and download visual diagnostic reports.

##  Technologies Used
* Python 3.9+
* Ultralytics YOLOv8
* Streamlit
* PyTorch & OpenCV
